# Public Datasets 
*Note: Some datasets have been amended to ensure they are in-line with the group dataset*

- 01-Run-Or-Walk: Malyi, V. (n.d.) Run or Walk. Kaggle. Available at: https://www.kaggle.com/datasets/vmalyi/run-or-walk (Accessed: 28 September 2025).

- 02-Kosovan-Walk, 03-Kosovan-Run: Kosovan, O. (2019) Data for activity recognition. Kaggle. Available at: https://www.kaggle.com/datasets/kosovanolexandr/data-for-activity-recognition (Accessed 14 October 2025).

- 04-KU-HAR: Sikder, N. (n.d.) KU-HAR: Human Activity Recognition Dataset (v 1.0). Kaggle. Available at: https://www.kaggle.com/datasets/niloy333/kuhar?resource=download (Accessed 14 October 2025).

- GB-Train-Stations-Maps: Marília, P. Kaggle. Available at: https://www.kaggle.com/code/mpwolke/european-train-stations-maps/input (Accessed 24 October 2025)
